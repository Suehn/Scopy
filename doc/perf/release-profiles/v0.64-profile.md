# v0.64 - Performance Profile

日期：2026-03-25

环境：
- `hw.model=Apple M3`
- macOS 26.4（25E241）
- Xcode 26.2（17C52）
- Swift 6.2.3
- 快照库：`perf-db/clipboard.db`

## 结论（摘要）

- **full-index warm-load 已从多秒级收敛到亚秒级**：最新 release 证据中 warm-load 从上一轮基线 `3049.715ms` 下降到 `612.389ms`，约提升 79.9%。
- **磁盘缓存命中路径现在可解释**：warm-load summary 会输出 phase、counter 与 reason，可直接区分 `disk_cache_hit`、`database_rebuild`、`checksum_mismatch` 等原因。
- **内存峰值仍在可接受范围内**：当前 peak RSS 为 `219.05MB`；虽然高于上一轮 `186.56MB`，但已换来明显更低的冷启动等待时间。
