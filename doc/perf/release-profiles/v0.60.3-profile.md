# v0.60.3 - Performance Profile

日期：2026-03-21

环境：
- `hw.model=Apple M3`
- macOS 26.4（25E241）
- Xcode 26.2（17C52）
- Swift 6.2.3
- 快照库：`perf-db/clipboard.db`（6421 items；≈ 148.6MB）

## 结论（摘要）

- **后端稳态没有明显回退**：service 路径的 `cm` / `数学` P95 相比 2026-02-28 baseline 分别下降 9.50% / 11.82%；engine 侧常用 query 基本维持在同量级。
- **full-index warm-load 现在有了明确基线**：同一套 release bench 产物已经把 warm-load latency 和 peak RSS 固化为可复跑证据，而不是只剩零散日志。
- **前端滚动帧稳定，drop ratio 和缩略图 decode 更好**：三类真实快照场景的 frame P95 都维持在 16.667ms；accessibility / text-bias 的 drop ratio 明显下降。
- **text-bias 当前变体没有产生 `text.metadata_ms` bucket 样本**：profile 保留缺失值，不把“0 samples”伪装成 0ms。
