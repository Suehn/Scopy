# Release History

This directory stores one immutable release note per shipped version.

## Rules

- Add new release notes here.
- Do not store current release metadata here; that belongs in [../../meta/release-current.yml](../../meta/release-current.yml).
- Do not use this directory as an automation API surface beyond the explicit `release_doc` path in metadata.
