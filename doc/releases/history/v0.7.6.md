# v0.7.6

## Summary

This release records the architecture and stability work behind the current Scopy frontend and Trellis development flow. It keeps user-facing clipboard, search, preview, export, settings, and selection behavior unchanged.

The main product change is a narrower history row architecture: row-ready presentation data now lives in `HistoryItemRowDescriptor`, while visible row thumbnail lifecycle sequencing lives in `HistoryRowThumbnailLifecycleScheduler`. The release also hardens Trellis `TASK_DIR` handling across Codex, Claude, and Cursor so exact Trellis subagents can continue from an explicit task directory even when their isolated session has no active task pointer.

## Key Changes

### Frontend Architecture

- Added `HistoryItemRowDescriptor` under `Scopy/Presentation` for row-ready title, metadata, thumbnail request, and display-model metric boundaries.
- Added `HistoryRowThumbnailLifecycleScheduler` for visible row thumbnail cache hits, cache-miss load priority, scroll-settle waits, cancellation, and path-tagged commit results.
- Kept SwiftUI-owned `@State`, `.task(id:)`, and final image commit guards in the row thumbnail views.
- Preserved the existing `row.display_model_ms` metric name so frontend profile continuity is not broken by the refactor.

### Trellis Agent Reliability

- Codex, Claude, and Cursor Trellis hooks now prefer explicit `TASK_DIR` plus verified `prd.md` over isolated no-current-task state.
- Exact Trellis agents now document and enforce the same TASK_DIR-first contract before loading PRD, JSONL context, and specs.
- Trellis meta documentation now warns against using generic/default/explorer agents as authoritative TASK_DIR workers.

### Frontend Profile Reliability

- `scripts/perf-frontend-profile.sh` now quits the Scopy bundle and clears leftover `Scopy` executable processes before the script, between baseline/current variants, and on exit.
- Frontend quality guidance now treats missing profile summaries after failed UI profile runs as no performance evidence, not as a partial benchmark.
