# v0.43.11 性能对比（v0.43.10 → v0.43.11）

**日期**: 2025-12-14

## 对比版本

- v0.43.10 → v0.43.11

## 测试环境

- 设备：MacBook Air（Mac15,12, 24 GB）
- 系统：macOS 15.7.2（24G325）
- 架构：arm64
- 配置：Debug
- 命令：`make test-perf`（`RUN_PERF_TESTS=1`）
- 电源：Low Power Mode disabled

## 测试用例变化

- 新增/更新：hover 预览与缩略图链路测试、Clipboard HTML 非 UTF-8 解析与 copyToClipboard 兜底测试等（详见 `doc/implementation/releases/v0.43.11.md`）。

## 性能回归说明

- 本版本主要改动在 UI 预览链路/缩略图生成方式与剪贴板 plain text 兜底；上述指标波动更可能来自运行环境噪声与 I/O 负载变化。
- `Regex 20k P95` 波动较大（+42%）：当前仍远低于阈值（200ms），暂不作为回归处理；如后续持续抬升，建议单独增加稳定样本并固定数据分布。

## 总结

- 关键搜索 SLO 仍满足；整体无明显性能回退。
