## 文档规范

每个性能对比文档必须包含：

1. **对比版本**: 明确标注 `vX.X → vY.Y`
2. **测试环境**: 硬件/系统/日期
3. **性能对比表格**:
   - 搜索性能 (5k/10k/25k/50k/75k)
   - 清理性能 (内联/外部/大规模)
   - 写入性能 (批量插入/去重)
   - 读取性能 (首屏/批量读取)
4. **测试用例变化**: 新增/删除的测试
5. **性能回归说明**: 如有回归，说明原因和是否需要修复
6. **总结**: 亮点、需关注项、SLO 达标情况

## 命名规范

```
vX.X-profile.md
```

例如：`v0.11-profile.md`, `v0.12-profile.md`
