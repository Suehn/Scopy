# v0.43.2 性能对比（v0.43.1 → v0.43.2）

**日期**: 2025-12-14

## 对比版本

- v0.43.1 → v0.43.2

## 测试环境

- 设备：MacBook Air（Apple M3, 24 GB）
- 系统：macOS 15.7.2（24G325）
- 架构：arm64
- 配置：Debug
- 命令：`make test-perf`（`RUN_PERF_TESTS=1`）
- 电源：Low Power Mode enabled（`pmset -g` 显示 `lowpowermode 1`）
- 备注：heavy 场景需 `RUN_HEAVY_PERF_TESTS=1` 才运行

## 性能对比表格

| 指标 | v0.43.1 | v0.43.2 |
|------|--------:|--------:|
| Fuzzy 5k items P95 | 8.49ms | 8.45ms |
| Fuzzy 10k items P95 | 79.41ms（Samples: 50；Low Power Mode enabled） | 78.50ms（Samples: 50；Low Power Mode enabled） |
| Disk 25k fuzzy P95 | 102.93ms（Samples: 50；Low Power Mode enabled） | 104.57ms（Samples: 50；Low Power Mode enabled） |
| Bulk insert 1000 items | 140.62ms（≈7,112 items/s；Low Power Mode enabled） | 83.76ms（≈11,940 items/s；Low Power Mode enabled） |
| Fetch recent (50 items) avg | 0.11ms | 0.12ms |
| Regex 20k items P95 | 5.25ms | 5.20ms |
| Mixed content disk search（single run） | 7.56ms | 7.31ms |

## 测试用例变化

- 无（用例集合未变；本版本主要为 UI 滚动降载与搜索取消更及时）。

## 性能回归说明

- v0.43.1 与 v0.43.2 的采样均在 **Low Power Mode enabled** 环境下完成；指标改善/波动仍可能包含系统抖动，但更可作为同一电源策略下的对照参考。

## 总结

- v0.43.2 重点是“交互/功耗场景的确定性”（滚动期降载、搜索取消中断 DB 查询）；建议后续补充 Low Power Mode 下的滚动/渲染指标采样，形成可重复的 UI 性能基线。
