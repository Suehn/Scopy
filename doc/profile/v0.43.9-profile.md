# v0.43.9 性能对比（v0.43.8 → v0.43.9）

**日期**: 2025-12-14

## 对比版本

- v0.43.8 → v0.43.9

## 测试环境

- 设备：MacBook Air（Apple M3, 24 GB）
- 系统：macOS 15.7.2（24G325）
- 架构：arm64
- 配置：Debug
- 命令：`make test-perf`（`RUN_PERF_TESTS=1`）
- 电源：Low Power Mode enabled（`pmset -g` 显示 `lowpowermode 1`）

## 性能对比表格

| 指标 | v0.43.8 | v0.43.9 |
|------|--------:|--------:|
| Fuzzy 5k items P95 | 8.40ms | 8.41ms |
| Fuzzy 10k items P95 | 76.10ms（Samples: 50） | 76.89ms（Samples: 50） |
| Disk 25k fuzzy P95 | 103.79ms（Samples: 50） | 108.72ms（Samples: 50） |
| Bulk insert 1000 items | 83.63ms（≈11,957 items/s） | 82.99ms（≈12,050 items/s） |
| Fetch recent (50 items) avg | 0.11ms | 0.11ms |
| Regex 20k items P95 | 5.26ms | 5.31ms |
| Mixed content disk search（single run） | 7.47ms | 7.50ms |

## 测试用例变化

- 无（本版本为 Clipboard/Storage 的并发语义与 I/O 下沉；perf 用例集合未变）。

## 性能回归说明

- 本版本主要改动为“避免主线程阻塞”的 I/O/转码下沉；搜索性能数值的细微波动更可能来自系统抖动与采样噪声。

## 总结

- v0.43.9 重点是降低预览/回写剪贴板的交互延迟并修复 `ClipboardMonitor` stop/start 语义；性能数据用于基线记录与回归监控。
