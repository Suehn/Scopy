# v0.43.6 性能对比（v0.43.5 → v0.43.6）

**日期**: 2025-12-14

## 对比版本

- v0.43.5 → v0.43.6

## 测试环境

- 设备：MacBook Air（Apple M3, 24 GB）
- 系统：macOS 15.7.2（24G325）
- 架构：arm64
- 配置：Debug
- 命令：`make test-perf`（`RUN_PERF_TESTS=1`）
- 电源：Low Power Mode enabled（`pmset -g` 显示 `lowpowermode 1`）

## 性能对比表格

| 指标 | v0.43.5 | v0.43.6 |
|------|--------:|--------:|
| Fuzzy 5k items P95 | 8.26ms | 8.23ms |
| Fuzzy 10k items P95 | 75.82ms（Samples: 50） | 77.35ms（Samples: 50） |
| Disk 25k fuzzy P95 | 99.52ms（Samples: 50） | 110.59ms（Samples: 50） |
| Bulk insert 1000 items | 84.10ms（≈11,891 items/s） | 82.74ms（≈12,086 items/s） |
| Fetch recent (50 items) avg | 0.11ms | 0.11ms |
| Regex 20k items P95 | 5.32ms | 5.31ms |
| Mixed content disk search（single run） | 7.46ms | 7.91ms |

## 测试用例变化

- 无（本版本主要为图片 hover 预览交互路径优化；perf 用例集合未变）。

## 性能回归说明

- 本版本未改动搜索/存储热路径的算法结构；指标波动更可能来自系统抖动与电源策略下的采样噪声。
- hover 预览体感提升不直接体现在 `make test-perf` 指标中（属于 UI 交互路径）。

## 总结

- v0.43.6 的重点是 hover 图片预览更及时（预取 + ThumbnailCache 优先级）；性能数据作为参考记录，若需严格对比请在相同电源策略与相近系统负载下复测。
